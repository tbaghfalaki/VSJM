# VSJM package


> Run a Gibbs sampler for variable selection for joint modeling of longitudinal measurements and survival time.

> Here we provide a comprehensible detailed vignette to variable selection using VSJM package.

> We illustrate the inference by a part of the real data of 
 our submitted paper entitled as follows:

  > Bayesian variable selection for joint modeling of  longitudinal measurements and survival data.


**Getting Started**

  > In this vignette we used the following R packages
